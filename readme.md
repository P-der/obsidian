# Obsidian 知识库

统一管理沉淀的技术知识与学习笔记。

## 目录结构

```
├── 01-编程语言/      # 各类编程语言学习笔记
├── 02-数据库/        # SQL、NoSQL 等数据库知识
├── 03-计算机基础/     # 算法、数据结构、操作系统等
├── 04-读书笔记/       # 书籍阅读笔记与总结
├── 05-工具效率/       # 开发工具、工作流优化
├── Inbox/            # 临时收集箱，待整理内容
└── image/            # 笔记中引用的图片
```

## 使用规范

- **Inbox 模式**：随手记到 `Inbox/收集箱.md`，定期整理归档
- **双向链接**：用 `[[笔记名]]` 建立笔记间关联
- **渐进式整理**：先记再改，不追求一次性完美
- **定期备份**：Git 保持版本控制

## 分类说明

| 目录 | 内容 |
|------|------|
| 01-编程语言 | Node.js、C++、Python 等 |
| 02-数据库 | PostgreSQL、MySQL、Redis 等 |
| 03-计算机基础 | 算法、数据结构、操作系统、网络 |
| 04-读书笔记 | 技术书籍阅读笔记 |
| 05-工具效率 | IDE、Git、Shell 等工具技巧 |
